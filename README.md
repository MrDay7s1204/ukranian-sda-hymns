**🎵 Гімни Української Адвентистської Церкви**

Мобільний додаток для Android, що містить збірку духовних пісень та гімнів.

📋 Про проєкт

Це стабільна версія цифрового пісенника. Додаток створений для того, щоб полегшити доступ до духовних пісень у будь-якому місці та в будь-який час. Я доклав максимум зусиль для розробки, і хоча в коді можуть зустрічатися дрібні помилки (баги), додаток виконує свою основну функцію.
